# 🛠 주요기능
1. Todo 추가
- `+` 버튼을 누르고 내용을 적으면 todo를 추가할 수 있다
2. Todo 삭제 및 수정
- edit버튼을 누르고 수정내용을 적은 후 엔터를 누르면 해당 내용이 바뀐다
3. Todo 체크
- Todo를 실천한 후 빈 원을 누르면 초록으로 변해 체크할 수 있다
